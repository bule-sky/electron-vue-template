# electron-vue

## Project setup
```
npm install
```

## Project install electron-builder
```
vue add electron-builder
```
[https://nklayman.github.io/vue-cli-plugin-electron-builder/](https://nklayman.github.io/vue-cli-plugin-electron-builder/guide/#installation)
